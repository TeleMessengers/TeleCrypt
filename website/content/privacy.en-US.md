## Contact

If you contact us by email, the data you provide will be stored by us to process your inquiry. We will delete the data
as soon as storage is no longer required or restrict processing if there are legal retention obligations.

## Hosting

If you use services within a browser on our site, we collect the following data in what are known as log files, which
your browser transmits:

- IP address
- Date and time of the request
- Time zone difference to Greenwich Mean Time
- Content of the request
- HTTP status code
- Amount of data transferred
- Website from which the request originates
- Information about browser and operating system.

This is necessary to display our website and to ensure stability and security. This corresponds to our legitimate
interest within the meaning of Art. 6 Abs. 1 S. 1 lit. f DSGVO.

The data will be deleted after 90 days.

The processing of the data specified in this section is neither legally nor contractually required. The functionality of
the website is not guaranteed without this processing.

## Matrix Client

If you use a Matrix Client provided by us, the following data will be processed and stored locally on your device:

- Matrix User ID
- Access token for using the Matrix Homeserver
- Keys for End-to-End Encryption (E2EE)
- Chat history
- Metadata necessary for the proper functioning of the app

The data is collected and temporarily stored solely to provide you with services via the Matrix protocol. All data
processing takes place within the Matrix Client on your device and is essential for its functionality.

You have full control over the data, which will be deleted if you uninstall the Matrix Client or log out.

We enable End-to-End Encryption (E2EE) by default to ensure that your communication is protected. Please note, however,
that not all rooms need to be encrypted.

### Note on the Matrix Homeserver

Please note that a Matrix Client communicates with a Matrix Homeserver to exchange messages and associated metadata.
The data processed and stored by the Homeserver is subject to the privacy policy of the Homeserver operator.
